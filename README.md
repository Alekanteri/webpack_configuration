## Scripts

### `yarn start`

Once collects files in the dist folder

### `yarn watch`

Monitors the changes in the main files, and rebuilds the resulting folder

### `yarn serve`

Creates a local server, and performs the same function as `yarn watch`. !No hot reload yet!
